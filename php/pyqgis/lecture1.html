Where to store custom scripts?

~/.qgis2/python/

* this directory is included in Python PATH

Creating scripts module folder

For purposes of this talk we will store all Python scripts to scripts directory

  1. Create scripts directory
  2. Create empty __init__.py file in this folder
~/.qgis2/python/scripts/__init__.py

Making the first script

  1. Create file named first.py in scripts directory
  2. Edit first.py so it is like:

# -*- coding: utf-8 -*-


class FeatureCounter():

    def __init__(self, iface):
        self._iface = iface

    def count(self):
        layer = self._iface.activeLayer()
        print len([x for x in layer.getFeatures()])

Executing the first script

* Load some layer files into qgis, so we can test script.

  1. Open Python Console [Ctrl + Alt + P]
  2. Import first.py module 
    from scripts import first
  3. Initialize the FeatureCounter class: 
    fc = first.FeatureCounter(iface)
  4. Execute count() method 
    fc.count()
    * change layers and reexecute count() to see different results.

Loading all shapefiles from folder

loader.py


# -*- coding: utf-8 -*-

from os import path
from glob import glob


class Loader:
    def __init__(self, iface):
        self._iface = iface

    def load(self, folder):
        for shp in glob(path.join(folder, "*.shp")):
            self._iface.addVectorLayer(shp, path.basename(shp), 'ogr')

Executing script in QGIS


from scripts import loader
l = loader.Loader(iface)
l.load("/path/to/some/shapefiles")

Scripting Tips & Tricks

Reloading python module

You need to reload python module after every code change, otherwise old code will still be used.

If you imported loader module by:

from scripts import loader

then loader is the name of the module and you can reload it by calling reload() function.

reload(loader)

[Python 3 reload() documentation]

Loading all modules in scripts

Edit __init__.py file in scripts folder so it contains: 

from os.path import dirname, basename, isfile
import glob
modules = glob.glob(dirname(__file__)+"/*.py")
__all__ = [basename(f)[:-3] for f in modules if isfile(f)]

Then we can load module all module in scripts inside QGIS with:


from scripts import *
